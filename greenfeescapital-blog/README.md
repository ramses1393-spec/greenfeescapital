## Stack

- Static HTML, no build step.
- Hosted on GitHub Pages.
- Monetization slots wired in invisibly (toggle `monetize-off` class on `<body>` to enable).
- SEO: JSON-LD WebSite + Organization, OpenGraph, Twitter Card, llms.txt, sitemap.xml.

## Files

- `index.html` — live page (today's drops)
- `assets/` — hero image, favicon, OG image
- `robots.txt` — welcomes AI crawlers (GPTBot, ClaudeBot, PerplexityBot, etc.)
- `llms.txt` — AI-optimized summary
- `sitemap.xml` — search engines
- `404.html` — out of bounds page
- `.nojekyll` — tells GitHub Pages to skip Jekyll processing
